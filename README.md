# IoT Sensor Monitor

Aplicación web para administrar sensores de IoT.

## Qué hace

- Login básico (simulado).  
- Visualiza sensores en tabla con filtros y orden.  
- Permite crear, editar y borrar sensores.  
- Actualiza datos en tiempo real vía NATS.

## Tecnologías

- React 
- TypeScript
- Firebase (autenticación)  
- nats.ws (cliente NATS)
